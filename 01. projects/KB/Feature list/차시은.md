# Feature list 인터뷰 전사 — 차시은

인터뷰어: 안녕하세요 반갑습니다, 오늘 인터뷰는 약 20분 정도 소요될 예정입니다. 준비 되셨나요?

참여자: 네.

인터뷰어: 평소 주로 거래하는 주 거래 은행은 어디인가요?

참여자: 카카오뱅크요.

인터뷰어: 아, 카카오뱅크를 주로 이용하시는군요. 혹시 카카오뱅크를 주 거래 은행으로 사용하게 된 특별한 계기가 있으신가요?

참여자: 자연스럽게 체크카드와 교통카드 발급받으면서 사용하게 됐습니다.

인터뷰어: 말씀 감사합니다. 혹시 모바일 뱅킹앱도 많이 사용하시나요?

참여자: 네.

인터뷰어: 그렇군요. 평소에 모바일 뱅킹 앱을 켜시면 주로 어떤 용도로 가장 많이 사용하시나요?

참여자: 사용 내역 확인할 때 가장 많이 씁니다.

인터뷰어: 사용 내역을 확인하시는 과정에서 메뉴를 찾기 어렵거나 화면 구성이 복잡해서 불편함을 느끼셨던 적은 없으신가요?

참여자: 한 번 정도는 있었던 것 같습니다.

인터뷰어: 아, 그러셨군요. 어떤 상황에서 어떤 메뉴를 이용하시다가 불편함을 느끼셨는지 조금만 더 구체적으로 말씀해 주실 수 있을까요?

참여자: 개인정보 관련해서 확인하려고 했던 것 같습니다.

인터뷰어: 개인정보 관련 메뉴를 찾으실 때, 구체적으로 어떤 화면에서 어떤 버튼을 찾기 어려우셨는지, 혹은 메뉴 이름이 예상과 달랐는지 등 당시 상황을 조금 더 자세히 설명해 주실 수 있을까요?

참여자: 조금 더 직관적으로 있을 줄 알았는데 제 생각보다 숨겨져 있어서 찾기가 힘들었습니다.

인터뷰어: 모바일 뱅킹앱을 '목적'없이 그냥 심심해서 들어가서 둘러보기도 하시나요?

참여자: 아니요.

인터뷰어: 그렇다면 앱 내에 송금이나 조회 외에도 다양한 부가 기능들이 많이 있는데, 평소에 그런 기능들을 따로 살펴보거나 관심을 가져본 적은 없으신가요?

참여자: 거의 없습니다.

인터뷰어: 현재 사용중인 주거래 뱅킹 앱이 얼마나 '직관적으로 사용하기 좋은 앱'이라고 생각하시나요? 전혀 동의하지 않으시면 1점, 완전 동의하면 5점을 주시면 됩니다

참여자: 4점 주겠습니다.

인터뷰어: 4점을 주셨군요. 혹시 '직관적으로 사용하기 좋다'고 느끼셨을 때, 구체적으로 앱의 어떤 점을 떠올리며 그렇게 생각하셨나요?

참여자: 들어가자마자 통장 및 연결된 카드 사용 내역으로 바로 이동할 수 있어서 좋습니다.

인터뷰어: 사용 내역을 바로 확인할 수 있는 점이 정말 편리하시겠네요. 혹시 최근에 그 기능을 사용하시면서 특히 만족스러웠거나 기억에 남는 구체적인 상황이 있으셨나요?

참여자: 카드 실적을 다 채워야 하는데 채웠는지 안 채웠는지 바로 확인할 수 있어서 좋았습니다.

인터뷰어: 감사합니다, 그러면 지금부터는 조금 다른 주제로 넘어가 보겠습니다. 화면을 하나 보여드리도록 하겠습니다. 이 화면을 한 10초 정도 살펴봐주시고, 다 보셨으면 다 봤습니다 라고 말씀해주세요

참여자: 다 맞습니다.

인터뷰어: 오늘의 운세를 알아보고 싶다고 생각해주세요. 지금 화면에서 오늘의 운세를 찾아볼까요?

참여자: 찾았습니다.

인터뷰어: 찾으셨군요. 그럼 그 메뉴를 보셨을 때 '여기에 운세 기능이 있을 것이다'라는 확신이 어느 정도 드셨나요? 5점은 완벽하게 확신이 들었다, 1점은 전혀 확신이 들지 않았다는 의미로 점수를 말씀해 주세요.

참여자: 5점.

인터뷰어: 그렇게 확신을 가지고 찾으셨는데, 실제로 메뉴를 찾아 들어가는 과정 자체는 수월하셨나요?

참여자: 네.

인터뷰어: 찾으시는 과정이 수월하셨다니 다행입니다. 혹시 이 기능을 더 찾기 쉽게 만들기 위해 메뉴의 이름이나 위치, 혹은 디자인 등에서 개선되었으면 하는 점이 있으신가요?

참여자: 아니요.

인터뷰어: 신분증이 필요한데, 실물 신분증을 두고 와서 모바일 신분증을 이용하려 합니다. 신분증을 이용할 수 있는 메뉴를 찾아보세요.

참여자: 찾았습니다.

인터뷰어: 그 메뉴를 처음 보셨을 때, '여기에 모바일 신분증 기능이 있을 것이다'라는 확신이 어느 정도 드셨나요? 5점은 완벽하게 확신이 들었다, 1점은 전혀 확신이 들지 않았다는 의미로 말씀해 주세요.

참여자: 5점.

인터뷰어: 확신이 높으셨군요. 그렇다면 실제로 해당 메뉴를 찾아 들어가는 과정 자체가 평소 예상하셨던 흐름대로 수월하게 느껴지셨나요?

참여자: 네.

인터뷰어: 메뉴를 찾으시는 과정이 수월하셨다니 다행입니다. 혹시 이 기능을 더 찾기 쉽게 만들기 위해 문구나 위치, 혹은 디자인 등에서 개선되었으면 하는 부분이 있으신가요?

참여자: 아니요.

인터뷰어: 은행 지점 방문 예정이라 상담 예약을 미리 진행하려 합니다. 은행 창구 상담 예약을 하려면 어디에서 시작해야 할까요?

참여자: 모바일 창구 부분에서 찾으면 됩니다.

인터뷰어: 그 메뉴를 찾으실 때 '여기에 상담 예약 기능이 있을 것이다'라는 확신이 어느 정도 드셨나요? 5점은 완벽하게 확신이 들었다, 1점은 전혀 확신이 들지 않았다는 의미로 1점에서 5점 사이로 말씀해 주세요.

참여자: 5점

인터뷰어: 그렇게 확신을 가지고 메뉴를 찾아 들어가셨는데, 실제로 그 경로를 통해 상담 예약을 진행하는 과정은 전반적으로 수월하셨나요?

참여자: 네.

인터뷰어: 상담 예약을 더 수월하게 찾을 수 있도록 메뉴의 이름이나 위치, 혹은 화면 구성 등에서 개선되었으면 하는 점이 혹시 있으신가요?

참여자: 딱히 없습니다.

인터뷰어: 이번 주말 휴양림을 이용하려 합니다. 휴양림을 예약할 수 있는 메뉴를 한번 찾아봐주시겠어요?

참여자: 잘 못 찾겠습니다.

인터뷰어: 로그인 절차 없이 계좌 확인이 가능한 빠른 로그인 기능을 이용하고 싶습니다. 어디에서 시작해야 될것 같은지 한번 찾아봐주세요

참여자: 잘 모르겠습니다.

인터뷰어: 잘 해주고 계십니다, 비슷한 방식으로 몇개의 시나리오를 더 여쭤보고 싶습니다. 내 계좌의 비밀번호를 변경하고 싶습니다. 어디로 가야 계좌 비밀번호를 번경할 수 있을지 찾아봐 주세요

참여자: 아무래도 계좌 관리로 가야 될 것 같습니다.

인터뷰어: 그 메뉴를 보셨을 때 '여기에 비밀번호 변경 기능이 있을 것이다'라는 확신이 어느 정도 드셨나요? 5점은 완벽하게 확신이 들었다, 1점은 전혀 확신이 들지 않았다는 의미입니다.

참여자: 4점.

인터뷰어: 방금 계좌 관리 메뉴를 통해 비밀번호 변경 기능을 찾으셨는데, 실제로 그 경로를 따라가시는 과정이 생각하셨던 것만큼 수월하셨나요?

참여자: 네.

인터뷰어: 방금 말씀하신 기능을 찾으실 때, 메뉴의 이름이나 위치, 혹은 화면 구성 등에서 조금 더 찾기 쉽게 개선되었으면 하는 부분이 있으셨나요?

참여자: 아니요.

인터뷰어: 자녀들을 위해서 자녀 입출금 통장을 개설하고 싶다면, 어디로 가서 시작을 해야할 지 찾아봐주세요

참여자: 어, 패밀리 뱅킹 일 것 같습니다.

인터뷰어: 패밀리 뱅킹 메뉴를 선택하셨는데, 그 메뉴를 보셨을 때 '여기에 자녀 통장 개설 기능이 있을 것이다'라는 확신이 5점 만점에 몇 점 정도 드셨나요? 5점은 완벽하게 확신이 들었다는 뜻이고, 1점은 전혀 확신이 들지 않았다는 의미입니다.

참여자: 5점.

인터뷰어: 방금 패밀리 뱅킹 메뉴를 통해 자녀 통장 개설 기능을 찾아보셨는데, 실제로 그 경로를 따라가면서 기능을 찾는 과정이 얼마나 수월하셨나요?

참여자: 수월했습니다.

인터뷰어: 찾으시는 과정이 수월하셨다니 다행입니다. 혹시 이 기능을 더 찾기 쉽게 만들기 위해 메뉴의 이름이나 위치, 혹은 화면 구성 등에서 개선되었으면 하는 점이 있으신가요?

참여자: 없습니다.

인터뷰어: 이체 한도를 변경하고 싶다면 어떤 메뉴로 가야할까요?

참여자: 이체 관리로 가면 될 것 같습니다.

인터뷰어: 그 메뉴를 선택하셨을 때, '여기에 이체 한도 변경 기능이 있을 것이다'라는 확신이 어느 정도 드셨나요? 5점은 완벽하게 확신이 들었다, 1점은 전혀 확신이 들지 않았다는 의미로 1점에서 5점 사이로 말씀해 주세요.

참여자: 5점.

인터뷰어: 그 메뉴를 통해 실제로 이체 한도 변경 기능을 찾아가는 과정은 전반적으로 수월하셨나요?

참여자: 네.

인터뷰어: 혹시 이 기능을 더 찾기 쉽게 만들기 위해 메뉴의 이름이나 위치, 혹은 화면 구성 등에서 개선되었으면 하는 점이 있으신가요?

참여자: 크게 없습니다.

인터뷰어: 통장사본을 제출해달라는 요청을 받아서, 통장 사본이 필요합니다. 어디서 발급 가능할 것 같으신가요?

참여자: 음 신분증명 사실 잘 모르겠습니다.

인터뷰어: 찾기 어려우셨군요. 혹시 이 기능을 더 쉽게 찾을 수 있도록 메뉴의 위치나 이름, 혹은 화면 구성에서 개선되었으면 하는 점이 있으신가요?

참여자: 통장 사본이라는 이름으로 존재를 하면 좋을 것 같고, 이미 존재한다면 좀 더 잘 보이는 위치에다가 놓는 게 나을 것 같습니다.

인터뷰어: 대출 이자를 지금 납부하고자 합니다. 이자를 납부할 수 있는 메뉴를 찾아봐주세요

참여자: 흠. 대출 테마에 들어가서 그 안에서 찾아야 되지 않을까요?

인터뷰어: 네, 대출 테마 안에서 찾으려고 하셨군요. 그 메뉴를 보셨을 때 '여기에 이자 납부 기능이 있을 것이다'라는 확신이 5점 만점에 몇 점 정도 드셨나요? 5점은 완벽하게 확신이 들었다, 1점은 전혀 확신이 들지 않았다는 의미입니다.

참여자: 2점.

인터뷰어: 그렇게 확신이 높지 않으셨는데, 실제로 대출 테마 안에서 해당 메뉴를 찾아가는 과정이 전반적으로 수월하게 느껴지셨나요?

참여자: 다양한 상품이 모여 있는 테마다 보니 그렇게까지 수월하지는 않을 것 같습니다.

인터뷰어: 그렇다면 이 기능을 더 찾기 쉽게 만들기 위해 메뉴의 이름이나 위치, 혹은 화면 구성 등에서 어떤 점이 개선되면 좋을지 의견을 주실 수 있을까요?

참여자: 아, 여기 상품 가입 관리 위쪽에 대출 테마가 있어서 거기로 들어갔으면 됐을 것 같은데, 그걸 제가 못 봤던 것 같습니다.

인터뷰어: 오랫동안 이체 기록이 없어, 이체 기능이 제한되버렸습니다. 이체를 다시 하기 위해, 제한을 풀어야 하는데요. 해제할 수 있는 메뉴를 찾아봐 주세요

참여자: 이체 관리

인터뷰어: 방금 '이체 관리' 메뉴를 선택해 주셨는데요. 이 메뉴를 누르기 전에 '여기에 이체 제한 해제 기능이 있을 것이다'라고 어느 정도 확신하셨나요? 5점은 완벽하게 확신했다, 1점은 전혀 확신이 들지 않았다는 의미입니다.

참여자: 5점.

인터뷰어: 확신하신 대로 메뉴를 찾으셨는데, 실제로 그 메뉴를 찾아 들어가는 과정은 예상했던 것만큼 수월하셨나요?

참여자: 네.

인터뷰어: 수월하게 찾으셨다니 다행입니다. 혹시 이 기능을 더 찾기 쉽게 만들기 위해 메뉴의 이름이나 위치, 혹은 화면 구성 등에서 개선되었으면 하는 점이 있으신가요?

참여자: 아니요, 괜찮을 것 같아요.

인터뷰어: 영업점 방문 없이 신규 계좌를 개설을 불허하는, '비대면 계좌 개설 안심차단 서비스를 이용해보고 싶어졌습니다. 어디로 가서 신청을 할 지 메뉴를 찾아봐주세요

참여자: 인증 보안에 들어가야 할 것 같습니다.
