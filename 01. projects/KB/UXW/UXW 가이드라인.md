## 평소 주로 거래하는 주 거래 은행은 어디인가요?
- 해당 은행과 관계가 깊어진 계기를 파악한다 
- 해당 은행과 관계가 더 깊어지게 된 과정과 특정 사건/이벤트가 있었는지 파악한다

## 말씀 감사합니다. 혹시 모바일 뱅킹앱도 많이 사용하시나요?

- 모바일 뱅킹 앱을 사용한다면
	- 모바일 뱅킹앱을 사용하는 목적은 주로 무엇인지 파악한다
	- 당시 목적 달성을 하는데 있어서, 사용사에 막히거나 불편한 지점은 없었는지 확인한다

## 모바일 뱅킹앱을 '목적'없이 그냥 심심해서 들어가서 둘러보기도 하시나요?

- 심심해서 둘러본다면
	- 왜 둘러보는지 파악한다
	- 둘러보면서 새롭게 알게된 정보나, 기능이 있었는지 파악한다
	- (IF) 만약 새롭게 알게된 정보나 기능이 있었다면, 어떤 정보 또는 기능이었는지 디테일을 파악한다
	- (IF) 만약 새롭게 알게된 정보나 기능이 있었다면, 그 새롭게 알게된 정보나 기능에 왜 특별히 관심을 가지고 들여다 보게 되었는지 이유를 파악한다
- 심심해서 둘러보지 않는다면
	- 뱅킹앱에 굉장히 많은 기능들이 있는데, 그런것들은 전혀 둘러보고나 관심이 가지 않는지 확인한다

## 현재 사용중인 주거래 뱅킹 앱이 얼마나 '직관적으로 사용하기 좋은 앱'이라고 생각하시나요? 전혀 동의하지 않으시면 1점, 완전 동의하면 5점을 주시면 됩니다

- '직관적인 사용'이라고 했을 때 구체적으로 어떤 것을 떠올리면서 답변을 주었는지 파악한다
- 해당 점수를 준 이유를 파악한다 
- 해당 점수를 주게 만든 구체적인 사례나, 경험이 있는지 파악한다

## "자산 확인하기"/"내 숨은 자산 찾기"/"자산 연결하기"/"내 현금 흐름 보기"라는 표현을 들었을 때, 뭔가 실제로 실행을 해보고 싶은 생각이 드시나요? 즉, 내 자산을 확인하고 싶으신지?

- 만약 눌러보고 싶다면
	- 실행을 해보고 싶은 이유를 파악
- 만약 눌러보고 싶지 않다면
	- 왜 눌러보고 싶지 않은지 이유를 파악

## "내 자산 확인하기"와 "내 숨은 자산 찾기" 중에 더 실행해보고 싶은 마음이 들게 하는 표현은 무엇인가요?
- "내 자산 확인하기"라고 답변을 한다면
	- 두 표현이, 같은 기능인데 표현만 다른것 같은지, 아예 다른 기능이라고 인식되는지 확인
	- 숨은 자산을 찾아준다는게 오히려 내가 놓치고 있던 돈을 찾아준다는 느낌같아서 더 실행해보고 싶을 것 같다는 의견도 있었는데, 참석자님께서는 다르게 생각하시나 보군요? 
- "내 숨은 자산 찾기"를 선택할 경우
	- 두 표현이, 같은 기능인데 표현만 다른것 같은지, 아예 다른 기능이라고 인식되는지 확인
	- "내 자산 확인하기"가 당장 내 자산을 볼 수 있어서 좋다는 의견이 있었는데, 참석자님은 의견이 다르신가 보군요? 
## "자산 연결하기"와 "내 현금 흐름 보기" 중에 더 실행해보고 싶은 마음이 들게 하는 표현은 무엇인가요?
- "자산 연결하기"라고 답변할 경우
	- 두 표현이, 같은 기능인데 표현만 다른것 같은지, 아예 다른 기능이라고 인식되는지 확인
	- 현금 흐름을 알아야 재정 관리가 되서 좋다는 의견들도 꽤 있었는데요, 참석자님의 생각은 다르신가보군요?
- 내 현금 흐름 보기 라고 답변할 경우
	- 두 표현이, 같은 기능인데 표현만 다른것 같은지, 아예 다른 기능이라고 인식되는지 확인
	- 자산 연결하기가 더 흩어져 있는 자산들이 모이는 것 같아서 좋다는 의견들도 있었는데, 참석자님의 의견은 다르신가 봅니다?
